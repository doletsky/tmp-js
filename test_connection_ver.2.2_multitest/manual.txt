Инструкция.

Текущая версия теста рассчитана на один тест на странице.

1. Файлы.
test_connection_data.js — данные теста — настройки отображения, содержимое ячеек (вопросы и ответы), связи между ними.
test_connection_jsAction.js - логика работы теста.
test_connection_cssView.css - внешний вид теста.
(файлы, перечисленные выше, для каждого теста уникальны)

test_connection.js - логика связи теста с контейнером.(для любого теста неизменен)

2. Создание теста.

В файле index.html внутри div class="interakt_zadanie" разместить <div class="job_exercises"></div> :
	<div class="interakt_zadanie">
        <div class="job_exercises"></div>
	</div>

Содержимое теста создается заполнением файла test_connection_data.js переменная test.(Коментарии в файле)
Тест визуально представляет из себя таблицу в которой одни ячейки содержат вопросы, другие - ответы.
Количество столбцов и колонок не ограничено.
Обязательно указать количество строк и колонок таблицы в row и col соответственно.
colTitle - заголовки колонок. Если заголовки не нужны, то можно просто исключить это свойство или оставить пустые {}.
jobData - содержимое ячеек.
dataType - тип ячейки - вопрос или ответ, 'q' или 'a' соответственно.
dataText - текст ячейки.
dataImg - изображение, если необходимо (если не нужно, можно просто исключить это свойство).
dataRight - правильные ответы для контроля, требуется, если ячейка содержит вопрос. Правильных ответов может быть несколько.
dataRight: {    /*правильные ответы*/
                            1:{         /*первый правильный ответ*/
                                row: 2, /*в строке 2*/
                                col: 2 /*в ячейке 2*/
                              },
                            2:{         /*второй правильный ответ*/
                                row: 3, /*в строке 3*/
                                col: 5 /*в ячейке 5*/
                            }
             }
